## 特色

- 支持下载
  - [ ] 歌手
  - [ ] 专辑
  - [x] 歌单
  - [x] 歌曲
  - [ ] 排行榜
- 支持下载高品质
  - [x] 臻品母带 (24Bit 192kHz)
  - [x] 臻品音质 (16Bit 44.1kHz)
  - [x] 臻品全景声 (16Bit 44.1kHz)
  - [x] flac (16Bit 44.1kHz~24Bit 48kHz)
  - OGG
    - [x] 640kbps
    - [x] 320kbps
    - [x] 192kbps
    - [x] 96kbps
  - MP3
    - [x] 320kbps
    - [x] 128kbps
  - AAC(M4A)
    - [x] 192kbps
    - [x] 96kbps
    - [x] 48kbps
